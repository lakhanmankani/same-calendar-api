Api Usage

Response:
{ "key": "9842824a468b7e0a1a01c9d802eb164c568ce3d86c1b7cfa0b80a7eab0379f3e" }

Get same calendar years

Request:
curl -X GET localhost:8080/api/same-calendar?key=<apikey>&year=2020&n=5
Response:
[2020, 2048, 2076, 2116, 2144]
